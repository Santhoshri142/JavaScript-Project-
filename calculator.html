BASIC CALCULATOR

Using this basic calculator we can do a basic arithmetic operations.

Steps to achieve those arithmetic oprations are given below....

  • We need two numbers or one operator to achieve arithmetic operations.
  • First click one number.
  • Click any operator.
  • Click a number to be added or multiplied with first number and to be subtracted or divided from the first number.
  • Finally the result will be displayed below the calculator.
